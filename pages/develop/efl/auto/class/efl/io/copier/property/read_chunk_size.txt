%%Copier read chunk size property, in bytes.%%

%%When a %%[[:docs:efl:auto:class:efl:io:copier:property:source|Efl.Io.Copier.source]]%% is flagged with %%[[:docs:efl:auto:interface:efl:io:reader:property:can_read|Efl.Io.Reader.can_read]]%% %%''true''%%, data will be read using %%[[:docs:efl:auto:interface:efl:io:reader:method:read|Efl.Io.Reader.read]]%% into an intermediate buffer of this size.%%
