===== Description =====

%%A wrapper socket doing SSL (Secure Sockets Layer).%%

%%Use this wrapper around an existing socket to do secure communication, a common use is to apply it to TCP sockets created with %%[[:develop:efl:auto:class:efl:net:dialer:tcp|Efl.Net.Dialer.Tcp]]%% or %%[[:develop:efl:auto:class:efl:net:server:tcp|Efl.Net.Server.Tcp]]%% created with "client,add" event.%%

//Since 1.19//
