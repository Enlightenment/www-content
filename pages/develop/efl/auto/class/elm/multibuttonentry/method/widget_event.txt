===== Description =====

%%Virtual function handling input events on the widget.%%

%%This method should return %%''true''%% if the event has been processed. Only key down, key up and pointer wheel events will be propagated through this function.%%

%%It is common for the event to be also marked as processed as in %%[[:develop:efl:auto:mixin:efl:input:event:property:processed|Efl.Input.Event.processed]]%%, if this operation was successful. This makes sure other widgets will not also process this input event.%%
