====== Efl.Canvas.Surface (mixin) ======

===== Description =====

%%Native surfaces usually bound to an externally-managed buffer.%%

%%The attached %%[[:develop:api:efl:canvas:surface:property:native_buffer|Efl.Canvas.Surface.native_buffer]]%% is entirely platform-dependent, which means some of this mixin's subclasses will not work (constructor returns %%''null''%%) on some platforms. This class is meant to be used from native code only (C or C++), with direct access to the display system or a buffer allocation system.%%
