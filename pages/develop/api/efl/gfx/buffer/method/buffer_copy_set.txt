===== Description =====

%%Set the pixels for this buffer by copying them, or allocate a new memory region.%%

%%This will allocate a new buffer in memory and copy the input %%''pixels''%% to it. The internal colorspace is not guaranteed to be preserved, and colorspace conversion may happen internally.%%

%%If %%''pixels''%% is %%''null''%%, then a new empty buffer will be allocated. If the buffer already had pixel data, the previous image data will be dropped. This is the same as %%[[:develop:api:interface:efl:gfx:buffer:method:buffer_managed_set|Efl.Gfx.Buffer.buffer_managed_set]]%%.%%

%%The memory buffer %%''pixels''%% must be large enough to hold %%''width''%% x %%''height''%% pixels encoded in the colorspace %%''cspace''%%.%%

%%%%''slice''%% should not be the return value of %%[[:develop:api:interface:efl:gfx:buffer:method:buffer_managed_get|Efl.Gfx.Buffer.buffer_managed_get]]%%.%%
