%%Control the format string for a given units label%%

%%If %%''NULL''%% is passed on %%''format''%%, it will make %%''obj''%%'s units area to be hidden completely. If not, it'll set the <b>format string</b> for the units label's text. Use a format string such as "%1.2f meters" for example.%%
