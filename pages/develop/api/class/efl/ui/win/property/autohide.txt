===== Description =====

%%Window's autohide state.%%

%%This property works similarly to %%[[:develop:api:class:efl:ui:win:property:autodel|Efl.Ui.Win.autodel]]%%, automatically handling "delete,request" signals when set to %%''trze''%%, with the difference that it will hide the window, instead of destroying it.%%

%%It is specially designed to work together with %%''ELM_POLICY_QUIT_LAST_WINDOW_HIDDEN''%% which allows exiting Elementary's main loop when all the windows are hidden.%%

%%%%[[:develop:api:class:efl:ui:win:property:autodel|Efl.Ui.Win.autodel]]%% and %%''autohide''%% are not mutually exclusive. The window will be deleted if both are set to %%''true''%%.%%
