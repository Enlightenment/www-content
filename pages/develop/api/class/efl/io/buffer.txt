===== Description =====

%%Generic In-memory buffer of data to be used as I/O.%%

%%This class offers both input and output, which can be used at the same time since %%[[:develop:api:interface:efl:io:reader:method:read|Efl.Io.Reader.read]]%% and %%[[:develop:api:interface:efl:io:writer:method:write|Efl.Io.Writer.write]]%% use different offsets/position internally.%%

%%One can get temporary direct access to internal buffer with %%[[:develop:api:class:efl:io:buffer:property:slice|Efl.Io.Buffer.slice]]%% or steal the buffer with %%[[:develop:api:class:efl:io:buffer:method:binbuf_steal|Efl.Io.Buffer.binbuf_steal]]%%.%%

%%A fixed sized buffer can be implemented by setting %%[[:develop:api:class:efl:io:buffer:property:limit|Efl.Io.Buffer.limit]]%% followed by %%[[:develop:api:class:efl:io:buffer:method:preallocate|Efl.Io.Buffer.preallocate]]%%%%

//Since 1.19//
