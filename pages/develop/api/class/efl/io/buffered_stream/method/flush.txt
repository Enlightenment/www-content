===== Description =====

%%Forces writing all pending data to destination.%%

%%It will return %%''true''%% if %%[[:develop:api:class:efl:io:buffered_stream:property:pending_read|Efl.Io.Buffered_Stream.pending_read]]%% drops to zero, %%''false''%% otherwise and more calls to flush must be made.%%

%%If the %%[[:develop:api:class:efl:io:buffered_stream:property:inner_io|Efl.Io.Buffered_Stream.inner_io]]%% is implements %%[[:develop:api:mixin:efl:io:closer|Efl.Io.Closer]]%% and it was closed, or the wrapper itself was closed, this function will do nothing and returns %%''true''%%.%%

%%@note this function may block the main loop execution until operations complete! This is bad for usability, as user interface or other operations may freeze. A better approach is to operate asynchronously and wait for "write,finished" event.%%
