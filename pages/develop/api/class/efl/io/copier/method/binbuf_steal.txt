===== Description =====

%%Steals the internal binbuf and return it to caller.%%

%%The buffer is then owned by caller, which should call eina_binbuf_free() when it's done.%%

%%Usually call this method when no %%[[:develop:api:class:efl:io:copier:property:destination|Efl.Io.Copier.destination]]%% is set, in this case you should wait for "done", "data" or "line" events and then call it to retrieve (and own!) the data.%%
