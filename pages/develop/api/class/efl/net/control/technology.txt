===== Description =====

%%A technology that enables network access points to be controlled.%%

%%The %%[[:develop:api:class:efl:net:control|Efl.Net.Control]]%% is composed of multiple technologies, each can be disabled/enabled with %%[[:develop:api:class:efl:net:control:technology:property:powered|Efl.Net.Control.Technology.powered]]%% property.%%

%%When powered, the technology will dynamically add and delete access points to be available in %%[[:develop:api:class:efl:net:control:property:access_points|Efl.Net.Control.access_points]]%%.%%

//Since 1.19//
