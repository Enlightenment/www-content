===== Description =====

%%This adds an item to the entry's contextual menu.%%

%%A longpress on an entry will make the contextual menu show up, if this hasn't been disabled with %%[[:develop:api:class:elm:entry:property:context_menu_disabled|Elm.Entry.context_menu_disabled.set]]%%. By default, this menu provides a few options like enabling selection mode, which is useful on embedded devices that need to be explicit about it, and when a selection exists it also shows the copy and cut actions.%%
