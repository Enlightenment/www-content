=== Rage Video and Media Player 2.0.1 Release ===
  * //2016-09-03 - by Raster//

This is an update release to [[/about-rage|Rage]] 2.0.0 with bug fixes like runaway thumbnailers fixed, using titles in metadata for movie title searches and dealing with some efreet "insane" responses, an improvement in thumb quality,

== Download ==
[[http://download.enlightenment.org/rel/apps/rage/rage-0.2.1.tar.gz|rage-0.2.1.tar.gz]]

=== What's New ===

  * Fix too many spawns of thumbnailer for seek bar thumbs
  * Avoid silly efreet dirs for Videos and fall back to ~/Videos then
  * Improve thumbnail quality for browser
  * Remove "silly" characters from video browser labels to be more readable
  * Use title metadata if the file is a move for poster search
