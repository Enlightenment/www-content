Note that a number of Enlightenment programs are switching to the [[http://mesonbuild.com/|Meson build system]].
To install Meson in Ubuntu, you can issue the following commands:
<code bash>
sudo apt-get install ninja-build python3-pip
</code>
Then:
<code bash>
pip3 install --user meson
</code>
If necessary, add:
<code bash>
export PATH=$HOME/.local/bin:$PATH
</code>
at the bottom of your ~/.bashrc file.

Meson [[https://git.enlightenment.org/apps/rage.git/tree/INSTALL|quick help]].

Then go to [[/docs/distros/]] to learn how to install the whole thing ;-)

<note tip>
Alternatively, if you do not want to delve into the many subtleties of the compilation process — while retaining a fully functional Ubuntu system after the build has finished — check out [[http://build-enlightenment.monsite-orange.fr/|batden's script]]: Covering Ubuntu 16.04 LTS and 17.10 with Wayland.</note>

== Wayland Support ==

Since Ubuntu has switched to using Wayland by default, things have become much easier.
Please refer to this [[https://git.enlightenment.org/core/enlightenment.git/tree/README.wayland|README]] for more information.

==== Binary Packages ====

The available packages in the official Ubuntu repositories are outdated.
The [[https://help.ubuntu.com/community/PPA|PPA]] below provides the latest //stable version// of E22.

To add this PPA to your system,
open a terminal and enter:

<code bash>
sudo add-apt-repository ppa:niko2040/e19
</code>

To install Enlightenment and Terminology (recommended):

<code bash>
sudo apt-get update
sudo apt-get install enlightenment terminology
</code>

To install EFL for development:

<code bash>
sudo apt-get update
sudo apt-get install libefl-dev
</code>

E22 //daily builds// PPA:

<code bash>
sudo add-apt-repository ppa:enlightenment-git/ppa
</code>

To install Enlightenment and Terminology (*-dev and *-dbg packages are also available):

<code bash>
sudo apt-get update
sudo apt-get install e20 terminology
</code>


Then log out, select Enlightenment on the login screen and
the configuration wizard will appear.
