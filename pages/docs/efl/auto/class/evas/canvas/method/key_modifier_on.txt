===== Signature =====

<code>
Evas.Canvas.key_modifier_on {
    params {
        @in keyname: string @nonull;
    }
}
</code>

===== C signature =====

<code c>
void evas_canvas_key_modifier_on(Eo *obj, const char *keyname);
</code>

===== Parameters =====

  * **keyname** //(in)// - %%The name of the modifier to enable.%%

===== Description =====

%%Enables or turns on programmatically the modifier key with name %%''keyname''%% for the default seat.%%

%%The effect will be as if the key was pressed for the whole time between this call and a matching evas_key_modifier_off().%%
