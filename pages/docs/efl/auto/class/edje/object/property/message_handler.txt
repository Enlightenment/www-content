===== Description =====

%%Set an Edje message handler function for a given Edje object.%%

%%For scriptable programs on an Edje object's defining EDC file which send messages with the send_message() primitive, one can attach handler functions, to be called in the code which creates that object (see @ref edcref "the syntax" for EDC files).%%

%%This function associates a message handler function and the attached data pointer to the object obj.%%
