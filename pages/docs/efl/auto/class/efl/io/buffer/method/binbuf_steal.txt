===== Description =====

%%Steals the internal buffer memory and returns it as a binbuf.%%

%%The returned memory must be freed with eina_binbuf_free().%%

%%On failure, for example a read-only backing store was adopted with %%[[:docs:efl:auto:class:efl:io:buffer:method:adopt_readonly|Efl.Io.Buffer.adopt_readonly]]%%, NULL is returned.%%
