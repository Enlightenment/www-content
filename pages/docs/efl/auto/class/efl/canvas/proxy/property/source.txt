===== Description =====

%%The source object for this proxy.%%

%%The proxy object will mirror the rendering contents of a given source object in its drawing region, without affecting that source in any way. The source must be another valid %%[[:docs:efl:auto:class:efl:canvas:object|Efl.Canvas.Object]]%%. Other effects may be applied to the proxy, such as a map (see %%[[:docs:efl:auto:mixin:efl:gfx:map|Efl.Gfx.Map]]%%) to create a reflection of the original object (for example).%%

%%Any existing source object will be removed after this call.%%

%%This property should be set as soon as creating a proxy object, otherwise the proxy will do nothing.%%
</note>

%%You cannot set a proxy as another proxy's source.%%
