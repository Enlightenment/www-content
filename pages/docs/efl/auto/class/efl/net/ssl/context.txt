===== Description =====

%%A SSL Context that is used to start a SSL socket wrapper.%%

%%The context will contain common configurations such as certificates, private keys, certificate revogation lists (CRLs), certificate authorities (CAs) and so on.%%

%%The method %%[[:docs:efl:auto:class:efl:net:ssl:context:method:setup|Efl.Net.Ssl.Context.setup]]%% must be called once before %%[[:docs:efl:auto:class:efl:object:method:finalize|Efl.Object.finalize]]%% in order to define the mandatory operational parameters.%%
