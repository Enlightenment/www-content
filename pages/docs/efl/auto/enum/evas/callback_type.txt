  * **image_resize** - %%Image size is changed%%
  * **device_changed** - %%Devices added, removed or changed on canvas%%
  * **axis_update** - %%Input device changed value on some axis%%
  * **canvas_viewport_resize** - %%Canvas viewport resized%%
