  * **does_not_exist** - %%The file pointed to did not exist%%
  * **permission_denied** - %%Permission to read the given file was denied%%
  * **resource_allocation_failed** - %%Resource allocation failed during the loading%%

  * **incompatible_file** - %%The file pointed to is incompatible, i.e., it doesn't match the library's current version's format%%
  * **unknown_collection** - %%The group/collection set to load from was not found in the file%%
  * **recursive_reference** - %%The group/collection set to load from had <b>recursive references</b> on its components%%
