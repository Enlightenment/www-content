===== Description =====

%%Generic interface for objects that can write data from a provided memory.%%

%%This interface allows external objects to transparently write data to this object and be notified if more data can be written or it's topped capacity.%%

%%Calls to %%[[:docs:efl:auto:interface:efl:io:writer:method:write|Efl.Io.Writer.write]]%%() may or may not block, that's not up to this interface to specify. The user can check with event "can_write,changed" or property %%[[:docs:efl:auto:interface:efl:io:writer:property:can_write|Efl.Io.Writer.can_write]]%% to known whenever a write could push more data.%%

//Since 1.19//
