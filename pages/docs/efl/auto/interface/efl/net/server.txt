===== Description =====

%%The basic server interface.%%

%%It will start serving and accepting clients once %%[[:docs:efl:auto:interface:efl:net:server:method:serve|Efl.Net.Server.serve]]%% is called and the "serving" event is dispatched.%%

%%When new clients are accepted, then "client,add" event is dispatched with a child object implementing %%[[:docs:efl:auto:interface:efl:net:socket|Efl.Net.Socket]]%% interface. These implement the standard %%[[:docs:efl:auto:interface:efl:io:reader|Efl.Io.Reader]]%%, %%[[:docs:efl:auto:interface:efl:io:writer|Efl.Io.Writer]]%% and %%[[:docs:efl:auto:mixin:efl:io:closer|Efl.Io.Closer]]%% interfaces, thus can be used with utilities such as %%[[:docs:efl:auto:class:efl:io:copier|Efl.Io.Copier]]%%.%%
