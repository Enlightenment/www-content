===== Description =====

%%The drag step increment.%%

%%Values for %%''dx''%% and %%''dy''%% are real numbers that range from 0 to 1, representing the relative size of the dragable area on that axis by which the part will be moved.%%

%%This differs from %%[[:docs:efl:auto:interface:efl:ui:drag:property:drag_page|Efl.Ui.Drag.drag_page]]%% in that this is meant to represent a unit increment, like a single line for example.%%
