Class Efl.ModelProvider

EFL object that provide a model to all. You can use this when you would otherwise have to call Efl.Ui.IView.SetModel(Efl.IModel) on multiple widgets by registering this object using Efl.Object.ProviderRegister(Type, Efl.Object) on a parent that they all depends on.

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.
