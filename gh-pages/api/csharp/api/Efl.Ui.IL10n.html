Interface Efl.Ui.IL10n

Interface for all translatable text APIs. This is intended for translation of human readable on-screen text strings but may also be used in text-to-speech situations.

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.

Syntax
public interface IL10n : IWrapper, IDisposable

Methods

GetL10nText(out String)

A unique string to be translated. Often this will be a human-readable string (e.g. in English) but it can also be a unique string identifier that must then be translated to the current locale with dgettext() or any similar mechanism.

Setting this property will enable translation for this object or part.

UpdateTranslation()

Requests this object to update its text strings for the current locale. Currently strings are translated with dgettext, so support for this function may depend on the platform. It is up to the application to provide its own translation data.

This function is a hook meant to be implemented by any object that supports translation. This can be called whenever a new object is created or when the current locale changes, for instance. This should only trigger further calls to Efl.Ui.IL10n.UpdateTranslation() to children objects.
