Class Efl.Ui.Spotlight.Indicator

Object used by Efl.Ui.Spotlight.Container to render an indication of the active widgets's position among the rest of the container's widgets. An example would be Android's little dots in the home screen.

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.

Bind(Container)

This method is called the first time an Efl.Ui.Spotlight.Indicator is assigned to an Efl.Ui.Spotlight.Container, binding them together. This call can be used to setup the indicator part of the spotlight.

The Active View of the container (if there is any) will be informed to the indicator by a later call to Efl.Ui.Spotlight.Indicator.UpdatePosition(System.Double).

UpdatePosition(Double)

This method tells the indicator that Efl.Ui.Spotlight.Container.ActiveIndex has changed in the bound container. position ranges from -1 to the number of views in the bound container (Efl.IContainer.ContentCount()). Notice this allows moving to a position before the first view or past the last view, which might happen if the view is thumb-scrolled out-of-bounds. Indicators can choose to render these out-of-bounds positions or not.

Fractional values indicate positions in-between two views and should be respected to obtain a smooth transition.
