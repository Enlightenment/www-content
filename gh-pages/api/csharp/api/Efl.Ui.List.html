Class Efl.Ui.List

A scrollable list of Efl.Ui.Item objects, typically Efl.Ui.ListDefaultItem objects. Items are added using the Efl.IPackLinear interface.

The orientation (vertical or horizontal) of the list can be set with Efl.Ui.ILayoutOrientable.Orientation.

Items inside this widget can be selected according to the Efl.Ui.IMultiSelectable.SelectMode (object still in beta stage) policy, and the selection can be retrieved with Efl.Ui.IMultiSelectable.NewSelectedIterator (object still in beta stage).

Efl.Ui.List supports grouping by using Efl.Ui.GroupItem (object still in beta stage) objects. Group headers are displayed at the top or left side of the viewport if items belonging to the group are visible in the viewport.
