Struct Efl.Event

A parameter passed in event callbacks holding extra event parameters. This is the full event information passed to callbacks in C. (Since EFL 1.22)

Info

Extra event information passed by the event caller. Must be cast to the event type declared in the EO file. Keep in mind that: 1) Objects are passed as a normal Eo*. Event subscribers can call functions on these objects. 2) Structs, built-in types and containers are passed as const pointers, with one level of indirection. (Since EFL 1.22)
