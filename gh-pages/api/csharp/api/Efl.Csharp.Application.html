Class Efl.Csharp.Application

This represents the entry point for the EFL framework You can use this class to implement the 4 abstract methods which will then be called accordingly All subsystems of efl are booted up correctly when the abstract methods of this class are called.

Remarks

Calls to efl outside those efl-callbacks or outside the mainloop are not allowed and will lead to issues
