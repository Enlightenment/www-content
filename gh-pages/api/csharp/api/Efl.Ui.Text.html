AddContextMenuItem(String, String, Type, EvasSmartCb, IntPtr)

This adds an item to the entry's contextual menu. A longpress on an entry will make the contextual menu show up unless this has been disabled with Efl.Ui.Text.SetContextMenuDisabled(System.Boolean). By default this menu provides a few options like enabling selection mode, which is useful on embedded devices that need to be explicit about it. When a selection exists it also shows the copy and cut actions.

With this function, developers can add other options to this menu to perform any action they deem necessary.
