Interface Efl.Ui.IRangeDisplay

Interface that contains properties regarding the displaying of a value within a range. A value range contains a value restricted between specified minimum and maximum limits at all times. This can be used for progressbars, sliders or spinners, for example.

Syntax
public interface IRangeDisplay : IWrapper, IDisposable

Properties

RangeLimits

Set the minimum and maximum values for given range widget. If the current value is less than min, it will be updated to min. If it is bigger then max, will be updated to max. The resulting value can be obtained with Efl.Ui.IRangeDisplay.GetRangeValue().

The default minimum and maximum values may be different for each class.

Note: maximum must be greater than minimum, otherwise behavior is undefined.

Declaration
(double, double)RangeLimits { get; set; }
