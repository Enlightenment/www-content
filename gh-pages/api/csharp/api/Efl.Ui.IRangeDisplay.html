Interface Efl.Ui.IRangeDisplay

Interface that contains properties regarding the displaying of a range.

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.

Syntax
public interface IRangeDisplay : IWrapper, IDisposable

SetRangeLimits(Double, Double)

Set the minimum and maximum values for given range widget. Define the allowed range of values to be selected by the user.

If actual value is less than min, it will be updated to min. If it is bigger then max, will be updated to max. The actual value can be obtained with Efl.Ui.IRangeDisplay.GetRangeValue()

The minimum and maximum values may be different for each class.

Warning: maximum must be greater than minimum, otherwise behavior is undefined.
