Interface Efl.IContent

Common interface for objects that have a single sub-object as content. This is used for the default content part of widgets, as well as for individual parts through Efl.IPart. (Since EFL 1.22)

Syntax
public interface IContent : IWrapper, IDisposable

Properties

Content

Sub-object currently set as this object's single content. If it is set multiple times, previous sub-objects are removed first. Therefore, if an invalid content is set the object will become empty (it will have no sub-object). (Since EFL 1.22)
