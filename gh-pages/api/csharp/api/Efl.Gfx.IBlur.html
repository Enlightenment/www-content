Interface Efl.Gfx.IBlur

A simple API to apply blur effects. Those API's might use Efl.Gfx.IFilter internally. It might be necessary to also specify the color of the blur with Efl.Gfx.IColor.GetColor(out System.Int32, out System.Int32, out System.Int32, out System.Int32).

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.

Syntax
public interface IBlur : IWrapper, IDisposable

Properties

Grow

How much the original image should be "grown" before blurring. Growing is a combination of blur & color levels adjustment. If the value of grow is positive, the pixels will appear more "fat" or "bold" than the original. If the value is negative, a shrink effect happens instead.

This is can be used efficiently to create glow effects.
