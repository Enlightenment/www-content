Interface Efl.Ui.Focus.ILayer

This defines the inheriting widget as focus layer A focus layer is the uppermost one which received input and handles all focus related events for as long as it exists and is visible. It's NOT possible to escape this layer with focus movement.

Once the object is hidden or destroyed the focus will go back to the mainwindow, where it was before.

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.

Syntax
public interface ILayer : IWidgetFocusManager, IManager, IWrapper, IDisposable
