Class Efl.Ui.WidgetFactory

Efl Ui Factory that provides Efl.Ui.Widget. This factory is designed to build Efl.Ui.Widget and optionally set their Efl.Ui.Widget.Style if it was connected with Efl.Ui.IPropertyBind.PropertyBind(System.String, System.String) "style".

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.
