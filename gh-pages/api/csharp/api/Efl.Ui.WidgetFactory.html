Class Efl.Ui.WidgetFactory

Efl Ui Factory that provides Efl.Ui.Widget. This factory is designed to build Efl.Ui.Widget and optionally set their Efl.Ui.Widget.Style if it was connected with Efl.Ui.IPropertyBind.BindProperty(System.String, System.String) "style".
