Class Efl.Ui.Slider

Elementary slider class This lets the UI user select a numerical value inside the Efl.Ui.IRangeDisplay.GetRangeLimits(out System.Double, out System.Double) limits. The current value can be retrieved using the Efl.Ui.IRangeInteractive interface. Events monitoring its changes are also available in that interface. The visual representation of min and max can be swapped using Efl.Ui.ILayoutOrientable.Orientation. Normally the minimum of Efl.Ui.IRangeDisplay.GetRangeLimits(out System.Double, out System.Double) is shown on the left side, the max on the right side.

RangeLimits

Set the minimum and maximum values for given range widget. If the current value is less than min, it will be updated to min. If it is bigger then max, will be updated to max. The resulting value can be obtained with Efl.Ui.IRangeDisplay.GetRangeValue().

The default minimum and maximum values may be different for each class.

Note: maximum must be greater than minimum, otherwise behavior is undefined.

SteadyEvent

Called when the widget's value has changed and has remained unchanged for 0.2s. This allows filtering out unwanted "noise" from the widget if you are only interested in its final position. Use this event instead of Efl.Ui.IRangeDisplay.ChangedEvent if you are going to perform a costly operation on its handler.
