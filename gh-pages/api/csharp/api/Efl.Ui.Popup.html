Class Efl.Ui.Popup

A styled container widget which overlays a window's contents. The Popup widget is a theme-capable container which can be used for various purposes. Regular contents can be set using the Efl.IContent interface, or basic scrollable contents can be set through the Efl.Ui.IWidgetScrollableContent (object still in beta stage) mixin API. For contents which should be scrollable but require more fine-grained tuning, it may be necessary for users to set up and provide their own scroller object such as Efl.Ui.Scroller.

A Popup widget will create an overlay for the window contents. This overlay is an Efl.Ui.PopupPartBackwall object, which provides functionality for passing events through to the main window while the Popup is active as well as the ability to set background images for the Popup.

By default, a Popup is positioned by the user through the Efl.Gfx.IEntity.Position property. This behavior can be altered by using the Efl.Ui.Popup.Align and Efl.Ui.Popup.Anchor properties. Setting the Efl.Gfx.IEntity.Position property directly will unset both the Efl.Ui.Popup.Align and Efl.Ui.Popup.Anchor properties, and vice versa.

By default, a Popup will size itself based on the minimum size of its contents through the Efl.Gfx.IHint interface. A Popup will never size itself smaller than the minimum size of its contents, but by manually setting the Efl.Gfx.IEntity.Size property or the Efl.Gfx.IHint.HintSizeMin property, a larger size can be specified.

Users can set a given Popup widget to close automatically after a specified time using the Efl.Ui.Popup.ClosingTimeout property.

For a Popup with a more specialized purpose, see Efl.Ui.AlertPopup.

Properties

Align

The align property specifies a Popup's current positioning relative to its anchor. When set, this property will override any user-provided value for the widget's Efl.Gfx.IEntity.Position property.

Declaration
public PopupAlign Align { get; set; }
Property Value
Type Description
Efl.Ui.PopupAlign

Alignment of the Popup relative to its anchor. The default is Efl.Ui.PopupAlign.None.

AlignPriority

This is the priority in which alignments will be tested using the anchor object if the value of Efl.Ui.Popup.Align is determined to be invalid. If a given alignment would result in the popup being partially or fully outside the visible area of the window, it is deemed invalid, and the next alignment is tested until either the priority list is exhausted or a usable alignment is found. An alignment will also be deemed invalid if the popup occludes its anchor object, except if Efl.Ui.PopupAlign.Center is specified.

Declaration
public (PopupAlign, PopupAlign, PopupAlign, PopupAlign, PopupAlign)AlignPriority { get; set; }
Property Value
Type Description
System.ValueTuple<Efl.Ui.PopupAlign, Efl.Ui.PopupAlign, Efl.Ui.PopupAlign, Efl.Ui.PopupAlign, Efl.Ui.PopupAlign>

First alignment. The default is Efl.Ui.PopupAlign.Top.

Anchor

The anchor object is the reference object for positioning a Popup using the Efl.Ui.Popup.Align and Efl.Ui.Popup.GetAlignPriority(out Efl.Ui.PopupAlign, out Efl.Ui.PopupAlign, out Efl.Ui.PopupAlign, out Efl.Ui.PopupAlign, out Efl.Ui.PopupAlign) properties. A Popup will recalculate its alignment relative to its anchor and change its position when: - the anchor object is moved (unless the anchor is a window) - the anchor object is resized - the Popup is resized - the parent window is resized

If Efl.Ui.Popup.GetAnchor() returns NULL, the anchor is the parent window of the Popup. If the anchor object is set to NULL, the Popup will no longer recalculate its alignment or change its position under any circumstance. If the Popup is moved by using Efl.Gfx.IEntity.SetPosition(Eina.Position2D), anchor is set NULL.

ClosingTimeout

The closing_timeout property is the time after which the Popup widget will be automatically deleted. The timer is initiated at the time when the popup is shown. If the value is changed prior to the timer expiring, the existing timer will be deleted. If the new value is greater than $0, a new timer will be created.

Declaration
public double ClosingTimeout { get; set; }
Property Value
Type Description
System.Double

If greater than $0, the Popup will close automatically after the value in seconds. The default is to not automatically delete the Popup.
