Interface Efl.Ui.IScrollbar

Interface used by widgets which can display scrollbars, enabling them to contain more content than actually fits inside them.

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.

Syntax
public interface IScrollbar : IWrapper, IDisposable
