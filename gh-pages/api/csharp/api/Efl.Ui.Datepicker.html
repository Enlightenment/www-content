Class Efl.Ui.Datepicker

Datepicker widget This is a widget which allows the user to pick a date using internal spinner. User can use the internal spinner to select year, month, day or user can input value using internal entry.

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.
