Class Efl.ContainerModel

Class used to create data models from Eina containers. Each container supplied represents a series of property values, each item being the property value for a child object.

The data in the given containers are copied and stored internally.

Several containers can be supplied and the number of allocated children is based on the container of the largest size.

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.
