Interface Efl.Ui.IDrag

Common interface for draggable objects and parts.

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.

Syntax
public interface IDrag : IWrapper, IDisposable

Properties

DragDir

Determines the draggable directions (read-only). The draggable directions are defined in the EDC file, inside the "draggable" section, by the attributes x and y. See the EDC reference documentation for more information.

Property Value
Type Description

MoveDragPage(Double, Double)

Moves the draggable by dx,dy pages. This moves the draggable by dx,dy pages. The increment is defined by Efl.Ui.IDrag.GetDragPage(out System.Double, out System.Double).

dx and dy can be positive or negative numbers, integer values are recommended.

Warning: Paging is bugged!

Declaration
bool MoveDragPage(double dx, double dy)
Parameters
Type Name Description

MoveDragStep(Double, Double)

Moves the draggable by dx,dy steps. This moves the draggable part by dx,dy steps where the step increment is the amount set by Efl.Ui.IDrag.GetDragStep(out System.Double, out System.Double).

dx and dy can be positive or negative numbers, integer values are recommended.
