Class Efl.Ui.Check

Check widget The check widget allows for toggling a value between true and false. Check objects are a lot like radio objects in layout and functionality, except they do not work as a group, but independently, and only toggle the value of a boolean between false and true.

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.
