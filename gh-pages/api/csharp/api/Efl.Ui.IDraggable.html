Interface Efl.Ui.IDraggable

Efl UI draggable interface

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.

Syntax
public interface IDraggable : IWrapper, IDisposable

Properties

DragTarget

Control whether the object's content is changed by drag and drop. If drag_target is true the object can be the target of a dragging object. The content of this object can then be changed into dragging content. For example, if an object deals with image and drag_target is true, the user can drag the new image and drop it into said object. This object's image can then be changed into a new image.

Declaration
bool DragTarget { get; set; }
Property Value
Type Description
System.Boolean

Turn on or off drop_target. Default is false.
