Enum Efl.Ui.WidgetOrientationMode

Widget orientation mode, or how the theme handles screen orientation. Note: Support for this feature is highly dependent on the theme in use. At the time of writing, the default theme for EFL does not implement support for orientation modes.

Fields

Name Description
Default

Default or automatic mode: if the widget's theme supports orientation, it will be handled automatically.
Disabled

No signal is sent to the widget's theme. Widget's theme will not change according to the window or screen orientation.
