Class Efl.Canvas.Gesture

EFL Gesture abstract class A gesture class defines a method that spcific gesture event and privides information about the gesture's type, state, and associated pointer information.

For cetain gesture types, additional methods are defined to provide meaningful gesture information to the user.

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.
