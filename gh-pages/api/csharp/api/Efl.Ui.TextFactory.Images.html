Class Efl.Ui.TextFactory.Images

Factory that creates images given key string The key can be either a full image path, or associated with one. The factory will fallback if key was not matches with an image, and try to load it as a full path.

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.
