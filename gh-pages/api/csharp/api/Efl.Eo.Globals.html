CreateWrapperFor(IntPtr, Boolean)

Creates a new wrapper for the given Eo id.

If the Eo was created from a non-generated class (i.e. C#-pure class), it returns the C# instance handle stored in the Eo's private data.

For generated-class Eo instance, we use reflection to get the correct C# type to re-wrap it.
