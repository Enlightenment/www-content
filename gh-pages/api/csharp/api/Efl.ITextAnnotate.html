CursorItemInsert(TextCursorCursor, String, String)

Inserts a object item at specified position. This adds a placeholder to be queried by higher-level code, which in turn place graphics on top of it. It essentially places an OBJECT REPLACEMENT CHARACTER and set a special annotation to it.
