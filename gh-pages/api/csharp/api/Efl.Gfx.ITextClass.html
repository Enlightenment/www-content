Interface Efl.Gfx.ITextClass

Efl Gfx Text Class interface

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.

Syntax
public interface ITextClass : IWrapper, IDisposable

Methods

DelTextClass(String)

Delete the text class. This function deletes any values for the specified text class.

Deleting the text class will revert it to the values defined by Efl.Gfx.ITextClass.GetTextClass(System.String, out System.String, out Efl.Font.Size) or the text class defined in the theme file.

Declaration
void DelTextClass(string text_class)
Parameters
Type Name Description
System.String text_class

The text class to be deleted.

GetTextClass(String, out String, out Size)

Get font and font size from edje text class. This function gets the font and the font size from text class. The font string will only be valid until the text class is changed or the edje object is deleted.
