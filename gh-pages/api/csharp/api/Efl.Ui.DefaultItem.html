Class Efl.Ui.DefaultItem

Default Item Class. This class defines the standard parts for most Efl.Ui.Item: Text, Icon and Extra. Placement and layout details for these parts are left to the extending classes like Efl.Ui.ListDefaultItem and Efl.Ui.GridDefaultItem which should normally be used. Text-related changes are mirrored to the text part for convenience. Content-related changes are mirrored to the icon part.
