Struct Efl.CallbackPriority

Callback priority. Range is -32k - 32k. The lower the number, the higher the priority. This is used to insert an event handler relative to the existing stack of sorted event handlers according to that priority. All event handlers always have a priority. If not specified Efl.Constants.CallbackPriorityDefault is to be assumed.

See Efl.Constants.CallbackPriorityBefore Efl.Constants.CallbackPriorityDefault Efl.Constants.CallbackPriorityAfter (Since EFL 1.22)
