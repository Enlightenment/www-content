Class Efl.Ui.Frame

Frame widget The Frame widget allows for collapsing and expanding the content widget by clicking on the frame label. the label and content can be set using text_set and content_set api.

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.
