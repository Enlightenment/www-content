Interface Efl.ITextStyle

Style to apply to the text A style can be coloring, effects, underline, strikethrough etc.

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.

Syntax
public interface ITextStyle : IWrapper, IDisposable
