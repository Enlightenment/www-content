Class Efl.Ui.WinSocket

An off-screen window to be displayed in a remote process. The window is rendered onto an image buffer to be displayed in another process' plug image object. No actual window is created for this type. The window contents can then be sent over a socket so that another process displays it inside a plug image.

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.
