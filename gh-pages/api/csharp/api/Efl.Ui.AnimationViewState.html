Enum Efl.Ui.AnimationViewState

State of animation view

Fields

Name Description
NotReady

Animation is not ready to play. (Probably, it didn't file set yet or failed to read file.
Pause

Animation has been paused. To continue animation call Efl.Ui.AnimationView.Resume (object still in beta stage). See Efl.Ui.AnimationView.Pause (object still in beta stage).
Play

Animation is playing. See Efl.Ui.AnimationView.Play (object still in beta stage).
PlayBack

Animation is playing back (rewinding). See Efl.Ui.AnimationView.PlayBack (object still in beta stage).
Stop

Animation view successfully loaded a file then readied for playing. Otherwise after finished animation or stopped forcibly by request. See Efl.Ui.AnimationView.Stop (object still in beta stage).
