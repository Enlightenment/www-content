Efl.Canvas.Object pivot

Pivot object for the center point. If the pivot object is NULL, then the object is rotated on itself.
System.Double cx

X relative coordinate of the center point. The left end is 0.0 and the right end is 1.0 (the center is 0.5).
System.Double cy

Y relative coordinate of the center point. The top end is 0.0 and the bottom end is 1.0 (the center is 0.5).
