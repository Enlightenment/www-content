Enum Efl.Pointer.Action

Pointer event type. Represents which kind of event this is. (Since EFL 1.19)

Fields

Name Description
Axis

Axis event (pen, stick, ...).
Cancel

Special event happening after a down if the up counterpart can not happen (eg. another window forcibly stole the focus).
Down

Mouse button or equivalent pointer pressed down. Always followed by up or cancel.
In

Mouse or pointer entered the object.
Move

Mouse or equivalent pointer moved.
None

Not a valid event, or nothing new happened (eg. when querying current state of touch points).
Out

Mouse or pointer exited the object.
Up

Mouse button or equivalent pointer released. See also cancel.
Wheel

Mouse wheel scroll, horizontally or vertically.
