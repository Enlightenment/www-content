Class Efl.Canvas.LayoutPartSwallow

Represents a SWALLOW part of an Edje object. Its lifetime is limited to one function call only, unless an extra reference is explicitely held.

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.
