Efl screen interface (Since EFL 1.22)

Syntax

ScreenRotation

Get the rotation of the screen. Most engines only return multiples of 90. (Since EFL 1.22)

Declaration

Property Value
Type Description
System.Int32

Screen rotation in degrees.

ScreenScaleFactor

Get screen scaling factor. This is the factor by which window contents will be scaled on the screen.

Note that on some display systems this information is not available and a value of 1.0 will be returned. (Since EFL 1.22)

Declaration
float ScreenScaleFactor { get; }
Property Value
Type Description
System.Single

The screen scaling factor.
