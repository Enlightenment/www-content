Interface Efl.Gfx.ISizeClass

Efl Gfx Size Class interface

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.

Syntax
public interface ISizeClass : IWrapper, IDisposable

Methods

DelSizeClass(String)

Delete the size class. This function deletes any values for the specified size class.

Deleting the size class will revert it to the values defined by Efl.Gfx.ISizeClass.GetSizeClass(System.String, out System.Int32, out System.Int32, out System.Int32, out System.Int32) or the size class defined in the theme file.

Declaration
void DelSizeClass(string size_class)
Parameters
Type Name Description
System.String size_class

The size class to be deleted.
