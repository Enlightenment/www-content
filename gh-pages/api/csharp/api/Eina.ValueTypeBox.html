Class Eina.ValueTypeBox

Boxing class for custom marshalling of ValueType enums.

As custom marshalling of enums (and other value types) is not supported, use use this boxing class as an intermediate at the Marshalling API level (like in marshall_type_impl.hh in the generator). User-facing API still uses Eina.ValueType normally.
