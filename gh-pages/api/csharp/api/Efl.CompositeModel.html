Class Efl.CompositeModel

Efl model for all composite class which provide a unified API to set source of data. This class also provide an Efl.IModel.GetProperty(System.String) "child.index" that match the value of Efl.CompositeModel.Index.

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.
