Interface Efl.ITextInteractive

This is an interface interactive text inputs should implement

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.

Syntax
public interface ITextInteractive : IText, ITextFont, ITextFormat, ITextStyle, IWrapper, IDisposable

Properties

Editable

Whether the entry is editable. By default text interactives are editable. However setting this property to false will make it so that key input will be disregarded.

Declaration
bool Editable { get; set; }
Property Value
Type Description
System.Boolean

If true, user input will be inserted in the entry, if not, the entry is read-only and no user input is allowed.

GetSelectionCursors(out TextCursorCursor, out TextCursorCursor)

The cursors used for selection handling. If the cursors are equal there's no selection.

You are allowed to retain and modify them. Modifying them modifies the selection of the object.
