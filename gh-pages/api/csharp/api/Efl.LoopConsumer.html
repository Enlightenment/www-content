Class Efl.LoopConsumer

An Efl.LoopConsumer is a class which requires one of the parents to provide an Efl.Loop interface when performing Efl.Object.FindProvider(Type). It will enforce this by only allowing parents which provide such an interface or NULL. (Since EFL 1.22)
