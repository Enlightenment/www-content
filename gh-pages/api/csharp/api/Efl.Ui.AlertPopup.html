Class Efl.Ui.AlertPopup

A variant of Efl.Ui.Popup which uses a layout containing a content object and a variable number of buttons (up to 3 total). An Alert_Popup is a popup which can be used when an application requires user interaction. It provides functionality for easily creating button objects on the popup and passing information about which button has been pressed to the button event callback.

Properties

Button

This property changes the text and icon for the specified button object. When set, the Alert_Popup will create a button for the specified type if it does not yet exist. The button's content and text will be set using the passed values.

Exactly one button may exist for each Efl.Ui.AlertPopupButton type. Repeated calls to set values for the same button type will overwrite previous values.

By default, no buttons are created. Once a button is added to the Popup using this property it cannot be removed.
