Interface Efl.ITextFormat

The look and layout of the text The text format can affect the geometry of the text object, as well as how characters are presented.

Remarks

This is a BETA class. It can be modified or removed in the future. Do not use it for product development.

Syntax
public interface ITextFormat : IWrapper, IDisposable
